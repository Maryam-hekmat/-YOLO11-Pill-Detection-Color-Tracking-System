# YOLO11 Pill Detection & Color Classification

**Real-time pharmaceutical capsule detection, tracking, and color classification** using **YOLO11** and **HSV analysis**.

---

## 🎯 Project Overview

An end-to-end computer vision system designed to detect, track, and classify colored capsules on a moving conveyor belt in real-time. This project simulates an industrial quality control scenario in the pharmaceutical industry.

### ✨ Key Features

- High-accuracy object detection using **YOLO11n**
- Robust multi-object tracking with **ByteTrack**
- Reliable color classification using **HSV color space**
- Real-time performance with clear bounding boxes and unique IDs
- Automatic color labeling (White-Blue, White-Pink/Red, etc.)

### 📊 Model Performance

- **Precision**: 0.9875
- **Recall**: 0.9824
- **mAP@0.5**: **0.9942** (99.42%)
- **mAP@0.5:0.95**: 0.9461

### 🛠️ Technologies Used

- **YOLO11n** (Ultralytics)
- **ByteTrack** for object tracking
- **OpenCV** for image processing
- **HSV Color Space** for robust color classification
- **Python 3.12**
