# YOLO11 Pill Detection & Color Classification System

**Real-time pharmaceutical capsule detection, tracking, and color classification** using **YOLO11 + ByteTrack + HSV Analysis**.

---

## 🎯 Project Overview

An end-to-end computer vision pipeline for automated quality control in the pharmaceutical industry. The system detects moving capsules on a conveyor belt, assigns unique tracking IDs, and accurately classifies their colors in real-time.

### ✨ Key Features
- High-accuracy object detection with **YOLO11n**
- Robust multi-object tracking using **ByteTrack**
- Reliable color classification using **HSV color space**
- Real-time performance with bounding boxes and unique IDs
- Automatic color labeling (White-Blue, White-Pink/Red, etc.)

### 📊 Model Performance
- **Precision**: 0.9875
- **Recall**: 0.9824
- **mAP@0.5**: **0.9942** (99.42%)
- **mAP@0.5:0.95**: 0.9461

### 🛠️ Technologies Used
- **YOLO11n** (Ultralytics)
- **ByteTrack** for object tracking
- **OpenCV** for image processing
- **HSV Color Space** for robust color classification
- **Python 3.12**
